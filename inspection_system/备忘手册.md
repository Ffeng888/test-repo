# 网络设施智能巡检系统 - 操作手册
## 一、连接机器狗
### 1.1 SSH连接（Windows CMD或PowerShell）
ssh unitree@192.168.50.25
# 密码: 你的密码
1.2 进入工作空间
cd ~/inspection_system
---
二、启动系统（需要4个终端）
【终端1】启动相机
# 连接机器狗后执行
ssh unitree@192.168.50.25
ros2 launch realsense2_camera rs_launch.py
检查: 看到 RealSense Node Is Up! 表示成功
---
【终端2】启动YOLO检测节点
# 新终端，SSH连接后执行
ssh unitree@192.168.50.25
cd ~/inspection_system
source /opt/ros/foxy/setup.bash
source install/setup.bash
爆T什么的内存错误就加上export LD_PRELOAD=/lib/aarch64-linux-gnu/libgomp.so.1
python3 ~/inspection_system/src/inspection_perception/inspection_perception/yolo_detector.py

检查: 看到 YOLO检测节点初始化完成! 和 相机内参已接收 表示成功
---
【终端3】启动Web服务器
# 新终端，SSH连接后执行
ssh unitree@192.168.50.25
cd ~/inspection_system
source /opt/ros/foxy/setup.bash
source install/setup.bash
python3 ~/inspection_system/src/inspection_viz/inspection_viz/web_server.py
检查: 看到 Web服务器已启动: http://0.0.0.0:5000 表示成功
---
【终端4】浏览器查看（在你的电脑上）
打开浏览器，访问：
http://192.168.50.25:5000
---
三、快速检查命令
检查节点是否运行
# 任意终端执行
ros2 node list
应该看到：
- /camera/camera (相机节点)
- /yolo_detector (检测节点)
- /web_server (Web服务器)
检查话题数据
# 检查相机话题
ros2 topic hz /camera/color/image_raw
# 检查检测结果
ros2 topic hz /detections
---
四、常见问题
问题1: 相机没启动
现象: ros2 topic hz /camera/color/image_raw 没有数据
解决: 重新插拔USB，或重启终端1
问题2: 节点崩溃
解决: 按顺序重启：
1. 先停终端3 (web_server): Ctrl+C
2. 停终端2 (yolo_detector): Ctrl+C  
3. 重新启动终端2和3
问题3: 浏览器打不开
检查:
# 在机器狗上测试web服务器
curl http://127.0.0.1:5000/api/status
如果返回JSON，说明服务器正常，检查电脑和机器狗是否在同一网络

五、一键启动脚本（推荐保存）
创建快速启动脚本：
cat > ~/start_inspection.sh << 'SCRIPT'
#!/bin/bash
# 巡检系统一键启动
cd ~/inspection_system
source /opt/ros/foxy/setup.bash
source install/setup.bash
echo "启动检测节点..."
python3 ~/inspection_system/src/inspection_perception/inspection_perception/yolo_detector.py &
sleep 3
echo "启动Web服务器..."
python3 ~/inspection_system/src/inspection_viz/inspection_viz/web_server.py &
sleep 2
echo "系统已启动!"
echo "浏览器访问: http://192.168.50.25:5000"
SCRIPT
chmod +x ~/start_inspection.sh
使用:
# 先启动相机（终端1）
ros2 launch realsense2_camera rs_launch.py
# 然后执行一键启动（终端2）
~/start_inspection.sh
