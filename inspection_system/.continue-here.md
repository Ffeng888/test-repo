### 模型输出格式
- Output0: `(batch, 300, 38)` - 预测结果
  - 38维 = 4(bbox_xywh) + 1(conf) + 1(cls) + 32(mask_coef)
- Output1: `(batch, 32, 160, 160)` - 分割proto（未使用）

### 坐标转换关键点
```python
# 当前实现 (可能有bug)
x1 = (cx - w/2 - x_offset) / scale
y1 = (cy - h/2 - y_offset) / scale

# x_offset/y_offset 是letterbox padding
# scale = 640 / max(orig_w, orig_h)
```
